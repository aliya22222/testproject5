## 2. Описание программы
Данное приложение — это генератор случайных задач, предназначенный для повышения продуктивности. Пользователь может выбрать случайную задачу из предопределённого списка, добавить новые задачи, а также фильтровать задачи по типам (учёба, спорт, работа). Все задачи сохраняются и загружаются из файла, что позволяет вести историю выполненных задач.

## 3. Возможности
- Генерация случайной задачи из списка.
- Ввод новых задач с указанием типа.
- Просмотр истории выполненных задач.
- Фильтрация задач по типам.
- Автоматическое сохранение истории в файл `tasks.json`.

## 6. Примеры использования
Пример 1:

    Нажмите «Сгенерировать задачу» — появится случайная задача, например: «Выучить 10 новых слов».
    Задача добавится в историю и отобразится в таблице.

Пример 2:

    Введите новую задачу, например: «Пойти на пробежку», выберите тип «спорт».
    Нажмите «Добавить задачу» — задача добавится в список и историю.
    Используйте фильтр «Спорт», чтобы увидеть только спортивные задачи.
